Toy Robot Simulator

ToyRobotApplication is the main class to start the robot sumulator.


Service Module
		Serices for Robot's movement and Direction
Model Module
		For Directios enum, Commands and application constants
Handler module
		Module to handle and parse input command and keep executing 
		application until user chooses to exit the application
		

Note:-
First commnd should always be PLACE or it will show error message
After place any command can be given which is valid for 5x5 tabel
